Jogo de labirinto desenvolvido no 1º ano do curso Técnico em Informática do IFRS – Campus Bento Gonçalves. A ideia é conduzir o bloco azul pelo caminho sem encostar nas paredes, avançando até a área verde para registrar a vitória.

A estrutura foi montada em HTML e SVG, permitindo criar cada parede, limite e elemento do cenário com posicionamento preciso. O CSS foi utilizado para ajustar cores, proporções e organização visual, deixando o trajeto bem definido e fácil de identificar.

A lógica do jogo foi implementada em JavaScript. O código acompanha o movimento do mouse, detecta colisões comparando as coordenadas do jogador com as paredes e reinicia a posição quando ocorre um erro. Também contabiliza vitórias, atualiza mensagens e controla toda a interação. O projeto serviu como prática de eventos, manipulação de atributos e integração entre HTML, CSS e JavaScript.
